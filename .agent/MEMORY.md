# Project Memory & Conventions

This file tracks project-specific preferences and technical requirements to ensure consistency.

## UI & Layout
- **Scroll Views**: Never use `SingleChildScrollView` directly. Always use `WindowsScrollView` (from `lib/widgets/widgets/windows_scroll.dart`) to ensure a consistent, smooth Windows-native feeling scroll experience.
- **Design Language**: Follow the "Instrument Panel" aesthetic (high density, technical, precise).
