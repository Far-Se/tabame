## Design Context

### Users
Power users and developers on Windows looking for a fast, distraction-free, and hyper-functional alternative to the standard taskbar.

### Brand Personality
**Minimal · Technical · Design-rich**
Tabame should feel like a high-performance instrument—precise and dense, but aesthetically elevated through thoughtful details.

### Aesthetic Direction
A blend of functional utility (inspired by PowerToys) and refined interaction design (inspired by Raycast). It is a "Modern Quick Menu" that balances dense information architecture with a state-of-the-art visual finish.

### Design Principles
1. **Bespoke Density**: Maximize information and action density without sacrificing legibility. No oversized padding; every pixel must earn its place on the screen.
2. **Refined Utility**: Functional elements should be beautiful through precise alignment, subtle color hierarchy, and technical-grade typography. 
3. **Zero-Friction Interactions**: Fast, reliable, and purposeful. The interface should feel "snapped" and reactive, avoiding unnecessary transitions that slow down the workflow.
4. **Contextual Cohesion**: Maintain a structural bridge between the technical nature of system-level utilities and a modern, modular design system.

### Visual Constraints
- **Typography**: Do not use `FontWeight.w900` for UI text or buttons; it looks too chunky. Use `w600` or `w700` for a technical, optically balanced feel.
- **Shadows**: Avoid aggressive hover glows or large blur-radius shadows (AI slop).
- **Depth**: Use extremely subtle LinearGradients (top-left to bottom-right) instead of glows. But use them only if necessary, like main Call-to-Action items.
  - Hover Standard: 4-stop sweep `[alpha: 0.08, alpha: 0.15, alpha: 0.20, alpha: 0.20]`
  - Idle Standard: 2-stop sweep `[alpha: 0.03, alpha: 0.08]`
- **Borders**: Use sharp, low-opacity borders with faded transitions for a high-utility instrument feel.
- **Optical Balance**: Ensure text weight is proportional to its size. Small labels should use medium weights for clarity rather than heavy weights which bleed into the surrounding pixels.
